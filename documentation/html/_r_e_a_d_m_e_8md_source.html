1 Machine Learing Final Project
2 ========
3 
4 The **goal** is to design a *labeled two class data set* of 10-dimensional vectors that has test set classification accuracy less than 60% on some popular classifiers.
5 However, our decision rule was designed such that it can perform with greater than 90% accuracy on the test set.

16 ### Data Set Generation
17 
18 `data_generation.py` is a *toolbox* written in Python for data set generation satisfying the problem specifications.
19 We generate a data set of 100,000 10-dimensional vectors (components), where 3 out them are relevant.
20 
21 Decision rules and data set generation using 10 components and discrete values (from 0 to 9) for supervised learning learning_driver.
22 The number of decision rules is reduced exponentially based upon the level of mate_learning (grouping of values) applied across the selected relevant components.
23 
24 ### Learning Method
25 
26 Supervised Learning method for binary classification based on mating (grouping of values) across the supplied data set of components and classification labels.
27 The learning algorithm is implemented in 'mate_learning.py`. It consists of exhausting all the possibilities as long as accuracy of correct classification can be increased.
28 
29 We define a `MateFinder` class for the particular binary classification problem based on mates (usually groups of 2 - pairs) across each component of a discrete-valued data set.
30 The `MateFinder` class provides procedures such as the removal of irrelevant components (selection of relevant components),
31 as well as essential learning (training and validation) and testing procedures from learned decision tables of pairs.
32 
33 Once a `MateFinder` object has been instantiated through the constructor, the interfacing methods employed by an external classifier are:
34 `learn_by_mate_discovery()` and `compute_prediction_accuracy()`
35 
36 The program driving the learning from data and estimating the accuracy of the learned decision rule on the testing set
37 is driven by `learning_driver.py`
